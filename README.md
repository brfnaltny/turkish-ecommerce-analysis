# Turkish E-Commerce Analysis

## Project Overview

This project analyzes customer purchasing behavior, sales performance, and delivery operations of a Turkish e-commerce company.

The objective is to transform raw transactional data into actionable business insights through data cleaning, exploratory data analysis, customer segmentation, SQL analysis, and interactive Power BI dashboards.

The project follows a complete analytics workflow:

Raw Data → Data Cleaning → Analysis → Insights → Business Decisions

---

## Dataset Information

The dataset used in this project was obtained from Kaggle and contains e-commerce transaction records from a Turkish online retail platform.

The dataset includes:

- Customer Information
- Order Information
- Product Categories
- Product Pricing
- Payment Methods
- Delivery Performance
- Customer Ratings
- Website Engagement Metrics

Key columns include:

- Customer_ID
- Order_ID
- Date
- Product_Category
- Total_Amount
- Payment_Method
- Device_Type
- Delivery_Time_Days
- Customer_Rating

## Business Questions

This project aims to answer the following business questions:

1. Which product categories generate the highest revenue?
2. Which cities contribute the most to sales?
3. Do returning customers spend more than new customers?
4. Who are the most valuable customers?
5. How much revenue is generated by VIP customers?
6. Can customers be segmented using RFM analysis?
7. Does delivery performance affect customer satisfaction?
8. Which customer segments require retention strategies?

## Key Insights

### Customer Insights
- VIP customers represent only 10% of customers but generate approximately 39% of total revenue.
- Revenue is distributed across a broad customer base rather than relying heavily on a small number of customers.
- Returning customers show similar average order values compared to new customers.

### Sales Insights
- Revenue performance differs significantly across product categories.
- Sales trends indicate seasonal purchasing behavior.

### Delivery Insights
- Delivery performance directly impacts customer satisfaction.
- Longer delivery times tend to be associated with lower customer ratings.
